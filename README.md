# 💸 실시간 가계부 정산 템플릿 생성기

> 매달 손쉽게 수입, 지출, 항목별 정산 내역을 작성하고 텍스트로 바로 추출해 주는 가계부 보조 웹 앱입니다.

---

## 📌 주요 기능 (Key Features)

- ⚡ **실시간 텍스트 생성**: 좌측에서 항목과 금액을 수정하면 우측에 공유용 정산 문구가 즉시 업데이트됩니다.
- ➕ **항목 자유 구성**: 대출, 공통 저축/생활비, 공과금, 개인별 지출(김가/이가 등) 항목을 자유롭게 추가/삭제할 수 있습니다.
- 📋 **클립보드 복사**: 클릭 한 번으로 생성된 정산 내역을 복사하여 카카오톡, 노션, 메모장 등에 붙여넣을 수 있습니다.
- 📥 **TXT 파일 다운로드**: 정산 완료된 내역을 `.txt` 파일로 다운로드하여 소장하거나 백업할 수 있습니다.
- 💾 **브라우저 로컬 저장/불러오기**: 작성 중인 내역을 브라우저(`localStorage`)에 안전하게 저장하고 나중에 다시 불러올 수 있습니다.

---

## 🛠️ 사용 기술 (Tech Stack)

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: Browser LocalStorage

---

## 🚀 사용 방법 (Usage)

1. 웹사이트 접속 후 **진행 월(Month)** 및 항목별 금액을 입력합니다.
2. 추가가 필요한 항목은 **`+ 항목 추가`** 버튼을 눌러 생성합니다.
3. 오른쪽 미리보기 영역에서 실시간으로 정돈된 문구를 확인합니다.
4. **`📋 전체 복사하기`** 또는 **`📥 TXT 다운로드`** 버튼을 눌러 결과물을 활용합니다.
5. 다음 달에도 사용하려면 **`💾 브라우저에 내용 저장`** 기능을 활용하세요!

---

## 🔒 데이터 보안 및 개인정보 (Privacy)

- 본 서비스는 별도의 서버나 데이터베이스를 사용하지 않습니다.
- 모든 입력 데이터는 사용자의 브라우저 내부(`localStorage`)에만 저장되므로 **개인정보 유출 걱정 없이 안심하고 사용**하실 수 있습니다.
